# expo-react-native-audio

엑스포를 사용하여 만든 오디오(뮤직) 앱입니다. 기존에 있는 오디오(뮤직) 앱과 매우 동일하며, 'expo-media-library'를 사용하여 오디오 타입을 불러와서 음악을 들을 수 있는 앱입니다.
